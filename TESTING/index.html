Automatizuotas vartotojų testavimas

Imituojama vartotojo elgsena ir tikrinama ar gauti rezultatai kurių tikėtasi

Pvz. Paspaudus mygtuką pasirodo dialogas.

Variantai

  • Tikros naršyklės
  • Headless našyklės

Galimybės

  • Viena naršyklė
  • Selenium gridas
  • Trečios šalies servisas

Testavimo aplinka

  1. Nodejs
  2. Selenium
  3. nightwatchjs

Testų vykdymas

  1. Atidarom du terminalus ir abiejuose:
  2. cd ~/uiacademy/TESTING/uiacademy-selenium
  3. pirmame: java -jar selenium.jar
  4. antrame: ./nightwatch --test tests/demo.test.js
