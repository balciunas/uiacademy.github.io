OOCSS (1)

  • Į CSS žiūrima kaip į lego detales iš kurių sudaroma svetainė
  • Jokio prisirišimo prie DOM
  • Tik klasės
  • Skatinamas stilių daugkartinis panaudojimas

OOCSS (2)

  • Pliusai
    • Atskirtos pagalbinės klasės
    • Aukštas stilių pakartotinis panaudojimas
    • Galima išmokti klases
    • Mažas skaičius specializuotų klasių
    • Mažas failo dydis
  • Minusai
    • Daug klasių ant vieno elemento
    • Reikia papildomų klasių norit papildomai apipavidalinti

CSS Technikos

CSS Gridai (1)

CSS klasių rinkinys skirtas dalinti turinį į stulpelius

Dažniausiai sudarytas iš eilutės klasės, bendros stulpelių klasės ir sulpelių dydžių klasių.

CSS Gridai (2) - CSS


.row:after, .row:before, .clearfix:before, .clearfix:after {
    content: "\0020";
    display: block;
    height: 0;
    overflow: hidden;
    clear: both;
}

.cell {
    float:left;
}

.size1of2 {
    width:50%;
}
.size1of4 { 
    width:25%;
}

CSS Gridai (3) - Struktūra


<div class="row">
    <div class="cell size1of2">A</div>
    <div class="cell size1of4">B</div>
    <div class="cell size1of4">C</div>
</div>
