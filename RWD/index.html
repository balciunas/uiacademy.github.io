Strategijos

Top down (Desktop first)

  1. Pradedam kai svetainė yra didžiausioje būsenoje (pvz. desktop 1440px)
  2. Mažinam langą tol kol svetainė nebetelpa į langą
  3. Atliekam reikalingus vizualinius pakeitimus
  4. Jei nepasiekėm mažiausios rezoliucijom, grįžtam į #2 žingsnį.

Bottom up (Mobile first)

  1. Pradedam kai svetainė yra mažiausioje būsenoje
    (pvz. iphone 5 320px)
  2. Didinam langą tol kol svetainė atrodo netinkamai
  3. Aliekam reikalingus vizualinius pakeitimus
  4. Jei nepasiekėm didžiausios rezoliucijos, grįžtam į #2 žingsnį.
